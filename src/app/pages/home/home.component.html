This is an HR application that caters to the capturing of employees information. The user of this application will be able to enter the following employee information:

  • First Name
  • Last Name
  • Employee Number
  • Address
    • Residential
    • Postal addresses
  • Contact details
    • landline number
    • mobile number
    • email address
    • social media addresses

The user may also view a list of all employees in the company, edit a single employee and delete a user where necessary.
